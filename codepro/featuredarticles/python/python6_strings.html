Python Strings

Strings in Python are sequences of characters used to store and manipulate text. They are immutable, meaning once a string is created, it cannot be modified.

Strings can be created using single, double, or triple quotes 'Hello World!' or "Hello World!"

Multiline String

A multiline string can be created by using triple quotes as follows:

Python

a ='''This is a multi line string 
used in python'''
print(a)

String Opeartions

  1. Concatenation :

    The process of joining two or more strings is known as concatenation.

    2. Python

    a ='Hello'
b ='World !'
print(a + " " + b)# return output: Hello World !
  2. Repetition :

    Repeating a string mutiple times using the * operator.You can repeat a string n number of times

    3. Python

    print("Python " * 3)
# returns output: Python Python Python
  3. Indexing:

    Strings in Python are arrays of bytes representing unicode characters.However, Python does not have a character data type, a single character is simply a string with a length of 1. Square brackets can be used to access elements of the string.

    Index values always starts from 0 and end at length of string -1 .

    Negative indexing is one of the useful feature in python in which the indexing takes from right side of the string start from -1 to ....-n .

    4. Python

    word = "Python"
print(word [0])# returns output: P
print(word [2])# returns output: t
print(word [-1])# returns output: n
  4. Slicing:
    5. Python provides special string operation called slicing which used to extract the range of characters over a string.

    Syntax :string_variable [start_index : end_index]

    Python

    word = "Python"
print(word [0 : 2])# returns output: Py
print(word [1 : 4])# returns output: yth
print(word [-3 : -1])# returns output: ho

    By using this slicing we can reverse a string easily in line of code in python which is as follows:

    Python

    word = "Python"
print(word [::-1])# returns output: nohtyP
  5. Length:
    6. Python provides a buit in method for finding the length of string which is len() returns the number of characters in a string including spaces and special characters.

    Python

    word = "Python"
print(len(word))# returns output: 6
word2 = "Pyt _hon"
print(len(word2))# returns output: 8

String Methods

  1. upper() and lower():
    2. These two methods are used to change the case of a string.upper() method is used to convert all the characters in string to upper case and lower() is used to convert to lower case.

    Python

    word = "Python"
print(upper(word2))# returns output: PYTHON  
print(lower(word2))# returns output: python
  2. strip():
    3. This method is used to remove the leading and trailing (ending) spaces from the a string.

    Python

    word = "  Python  "
print(strip(word))# returns output: Python
  3. split():
    4. This method is used to splits the string multiple parts based on the characters given where the split operation takes and returns the result values in a list format.

    Python

    word = "Python string concept"
print(word.split())# returns output: ['Python','string','concept'] 
word = "Python%string%concept"
print(word.split('%'))# returns output: ['Python','string','concept']
  4. join():
    5. This method is used to join multiple strings opposite to split() method .For example a list contains contains multiple strings the join() can join all those strings to a single string.

    Python

    words = ["Python" ,"string", "concept"]
print("-".join(words))# returns output: Python-string-concept
  5. find()andreplace():
    6. find() method is used to find the index of the first character of substring in the main string,replace() method is used to replace one string with another string.

    Python

    word = "Python Tutorial"
print(word.find("Tutorial"))# returns output: 7 
print(word.replace("Tutorial","Class"))# returns output: Python Class
  6. startswith()andendswith()
    7. These methods are used to check whether the string starts with specific character/string or ends with specific character/string respectively.

    Python

    word = "Python Tutorial"
print(word.startswith("Tutorial"))# returns output: False 
print(word.endswith("Tutorial"))# returns output: True

String Formatting

  1. old style (% operator):

    2. Python

    name = "john"
age = 30
message = "My name is %s and I am %d years old."%(name,age)
# output: My name is john and I am 30 years old.
  2. str.format()Method :

    3. Python

    message = "My name is {} and I am {} years old.".format(name,age)
# output: My name is john and I am 30 years old.
  3. F-Strings(python 3.6+)

    4. Python

    message = "My name is {name} and I am {age} years old."
# output: My name is john and I am 30 years old.

Use cases of Strings

  • Data Storage: Strings store textual data, such as names, addresses, and messages.
  • Data Parsing: Strings are used in reading and processing textual data from files or user input.
  • Web Devlopment:Generating and handling HTML, JSON, and XML.
  • Data Serialization: Converting data structures to strings for storage or transmission.
  • Logging and Debugging: Recording events and debugging information as strings.

Understanding string operations and methods is crucial for text processing and manipulation in Python.
