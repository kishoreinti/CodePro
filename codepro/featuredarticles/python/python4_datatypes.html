Python Data Types

In programming, data type is an important concept. Variables can store data of different types, and different types can do different things. Python provides a variety of data types to store and manipulate data efficiently.

  1. Numeric Types:
    • int: Used for Integers consists of positive,negative numbers including zero (e.g 1, 4, 0, -5).
    • float: Used for decimal or floating-point numbers (e.g 3.414, -0.428).
    • complex: Used for complex numbers with real and imaginary parts (e.g 3 + 4j, -1 + j).
  2. Sequence Types:
    • str: Represents a sequence of characters, used for text (e.g., "hello").
    • list: Represents an ordered, mutable (changable) collection (e.g., [1, 2, 3]).
    • tuple: Represents an ordered, immutable collection (e.g., (1, 2, 3)).
  3. Mapping Type:
    • dict: Represents a collection of key-value pairs, where keys are unique (e.g., {"name": "Alice", "age": 25}).
  4. Set Types:
    • set: Represents an unordered collection of unique elements (e.g., {1, 2, 3}).
    • list: Represents an immutable version of a set.
  5. Boolean Type:
    • bool: Represents True or False values
  6. Binary Types:
    • bytes: Represents Immutable sequence of bytes (e.g., b"hello").
    • bytearray: Represents Mutable sequence of bytes.
    • memoryview: Represents A memory-efficient way to access slices of binary data..
  7. None Type:
    • None: Represents the absence of a value. Only one instance exists: None.

Key Features

  • Dynamic Typing:
    • Python variables don’t require explicit type declaration. The type is determined at runtime.
  • Type Conversion:
    • Python allows explicit type conversion using functions like int(), float(), str(), etc.
