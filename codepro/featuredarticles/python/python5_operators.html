Operators in Python

In Python, operators are special symbols or keywords used to perform operations on variables and values. Python supports several types of operators, which can be broadly categorized as follows:

1. Arithmetic Operators

These operators perform basic mathematical operations.Let us declare two variables x and y and execute arithmetic operations on them.

x = 10
y = 20
  • Addition (+) : 
    print(x + y)  # adds x and y returns 30
  • Subtraction (-) : 
    print(x - y)  # subtracts y from x returns -10
  • Multiplication (*) : 
    print(x * y)  # multiplies x and y returns 200
  • Division (/) : 
    print(y / x)  # divides y by x returns 2
  • Modulus (%) : 
    print(y % x)  # returns the remainder of y divided by x which is 0
  • Floor Division (//) :

    Python

    x = 5
y = 2
print(x // y) # returns the floor value of x divided by y which is 2 remove the values after decimal point
  • Exponentiation (**) :

    Python

    x = 5
y = 2
print(x ** y) # Returns x raised to the power of y returns 25

3. Relational Operators

Relational operators are used to compare the values of two variables which are also known as comparision operators

  • == : (Equal to)
    • x == y# Returns True if x is equal to y
  • != : (Not Equal to)
    • x != y # Returns True if x is not equal to y
  • > : (Greater than)
    • x > y # Returns True if x is greater than y
  • < : (Less than)
    • x < y # Returns True if x is less than y
  • >= : (Greater than or equal to)
    • x >= y # Returns True if x is greater than or equals to y
  • <= : (Less than or equal to)
    • x <= y # Returns True if x is less than or equals to y

4. Logical Operators

These operators are used to combine two or more conditional statements (further topic)

  • and: Logical AND 
    x and y  # Returns True is x and y both are True
  • or: Logical OR 
    x or y  # Returns True if any one of  x and y are True
  • not: Logical NOT 
    not x  # Returns True if x is false

5. Bitwise Operators

Bitwise operators are used to perform operations on bits and comparing binary numbers.Before working with Bitwise operators you should have an overview on binary numbers and their operations.

  • & : Bitwise AND

    The & operator compares each bit and set it to 1 if both are 1, otherwise it is set to 0

    Python

    x = 2
y = 3
print(x & y)# returns the output 2
# binary of 2 is '10' and binary of 3 is '11'
# result of '10' & '11' is '10' which is binary of 2
  • | : Bitwise OR

    The | operator compares each bit and set it to 1 if one or both is 1, otherwise it is set to 0

    Python

    x = 2
y = 3
print(x | y)# returns the output 3
# binary of 2 is '10' and binary of 3 is '11'
# result of '10' | '11' is '11' which is binary of 3
  • ~ : Bitwise NOT

    The ~ operator inverts each bit (0 becomes 1 and 1 becomes 0).

    Python

    x = 2

print(~ x)# returns the output -3
# binary of 2 is '10' and ~ '0000000000000010' is '1111111111111101' which is binary of -3
  • << : Bitwise Left Shift

    The << operator inserts the specified number of 0's (in this case 2) from the right and let the same amount of leftmost bits fall off

    Python

    x = 2
print(x << 2)# returns the output 8
# binary of 2 is '10' then '10'<< 2 results 1000 which is binary of 8
  • >> : Bitwise Right Shift

    The >> operator moves each bit the specified number of times to the right. Empty holes at the left are filled with 0's

    Python

    x = 2
print(x >> 2)# returns the output 0
# binary of 2 is '10' then '10'>> 2 results 00 which is binary of 0

6. Identity Operators

Identity Operators are used to compare the memory location of two objects, not if they are equal, but if they are actually the same object, with the same memory location

  • is : Identity

    Python

    x = [1,2,3]
y = [1,2,3]
z = x
print(x is y)# returns False
print(x is z)# returns True
  • is not : Non-Identity

    Python

    x = [1,2,3]
y = [1,2,3]
print(x is not y)# returns True

7. Membership Operators

These operators are used to test for a membership in a sequence.

  • in : Membership

    Tests if a value is present in a sequence (such as a list, tuple, string..etc.,)

    Python

    x = [1,2,3]
print(1 in x)# returns True
  • not in : Non-Membership

    Tests if a value is not present in a sequence (such as a list,tuple,string..etc.,)

    Python

    x = [1,2,3]
print(1 not in x)# returns False

Conclusion

Each of these operators allows for a wide variety of operations and interactions with data in Python, enabling the creation of complex and efficient programs.
