Python Variables

A Variable in Python is a symbolic name that serves as a reference or pointer to an object. Variables are used to store data values that can be used and manipulated in a program. Python variables are dynamic, meaning you don't need to explicitly declare their type, as Python automatically determines it at runtime.

Key Features of Python Variables

  1. Dynamic Typing:
    • You don't need to declare the type of a variable explicitly.
    • The type is inferred from the value assigned.
    2. x=10# Integer
x="Hello"# String
x=3.45# Float
  2. Mutable and Immutable Types:
    • Mutable types: The value of the object can be changed (e.g., list, dict, set).
    • Immutable types: The value of the object cannot be changed (e.g., int, str, tuple).
  3. No Explicit Declaration:
    • Variables are created when a value is assigned to them.
    x = 10

Name Rules for Variables

  • A variable must start with a letter or an underscore (_).
  • A variable cannot start with a number.
  • A variable van only contain alphanumeric characters and underscores.
  • Reserved keywords like if, else, for, while, etc. should not be used.

Working With Variables

  1. Multiple Assignments:You can assign values to multiple variables in one line. 
    x,y,z = 10,20,30
# assigns x=10 y=20 z=30
  2. Reassignment:
    • A variable can be reassigned to a new value, even of a different type.
    x = 10
x = "Python Tutorial"# x is a string now
  3. Swapping Variables:
    • Python allows easy swapping of variables.
    x = 10
y = 20
x,y = y,x
# now x=20 and y=10  x and y values are swapped

Global and Local Variables

  • Global Variables:
    • Variables that are created outside of a function are known as global variables. Global variables can be used by everyone, both inside of functions and outside. Global variables are accessible throughout the program
  • Local Variables:
    • Variables that are created within the function are known as local variables. A local variable is accessible only within a function.

    If you create a variable with the same name inside a function, this variable will be local, and can only be used inside the function. The global variable with the same name will remain as it was, global and with the original value.

    Python

    x = "I am global variable"
def myfun():
    x = "I am local variable"
    print(x)
myfun()# Output: I am local variable
print(x)# Output: I am global Variable

    global keyword:

    Normally, when you create a variable inside a function, that variable is local, and can only be used inside that function. To create a global variable inside a function, you can use the global keyword.

    Python

    def myfun():
    global x 
    x = "I am local variable"
    
myfun()# Output: I am local variable
print(x)# Output: I am local Variable

Common Operations

  • Checking Data Type:
    • type keyword is used to return the data type of a variable

    Python

     x = 10
        printtype(x)# output : class 'int' 
        y = "Python Tutorial"
        printtype(y)# output : class 'string'
  • Deleting variables:
    • del keyword is used to delete a variable 
    del x
