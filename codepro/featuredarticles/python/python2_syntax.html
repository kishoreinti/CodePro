Python Syntax

Introduction

Python is a beginner-friendly programming language known for its clean and easy-to-read syntax.

Indentation in Python

Indentation refers to the spaces at the beginning of a code line. Where in other programming languages the indentation in code is for readability only, the indentation in Python is very important. Python uses indentation to indicate a block of code.

Python uses indentation (spaces or tabs) to define blocks of code, replacing curly braces {} or keywords like end.

Example:

# Correct Indentation
if age > 18:
    print("Adult")

Python returns an ERROR:INVALID SYNTAX if the indentation is not correct

# Incorrect Indentation
if age > 18:
print("Adult")

Case Senstivity

Python is a case-sensitive programming language. This means that identifiers (such as variable names, function names, and class names) that differ only in letter casing are treated as distinct. 

#both the variables are considered as two differnt ones
variable_1=10
VARIABLE_1=20

Comments

Comments in Python are used to explain code and make it more readable. They are ignored by the Python interpreter during execution.

Stay tuned for more tutorials as we dive deeper into Python's features and use cases. Happy coding!

Type of Comments

  1. Single Line Comments:

    Starts with #symbol

    #This is Single Line Comment
print('Python')#comment after a statement
  2. Multi Line Comments:

    There are no dedicated multi-line comment symbols in Python, but triple quotes (''' or """) are commonly used for this purpose.

    """
This is a multi-line comment.
You can write multiple lines of text here.
"""
print('Hello World!')
Why Use Comments?

  • Explain Logic: Clarify complex or non-obvious code logic.

  • Disable Code: Temporarily remove lines of code from execution.

  • Documentation: Provide instructions or describe the purpose of the code.
